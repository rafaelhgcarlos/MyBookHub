# MyBookHub 📚

**Organize seu backlog de livros de maneira simples e eficiente!**

## 📖 Sobre

O **MyBookHub** é um projeto de gerenciamento de livros, com foco em ajudar os usuários a organizar e acompanhar os livros que desejam ler. O objetivo é oferecer uma interface prática e intuitiva para adicionar, classificar e filtrar livros de maneira simples.

### Funcionalidades:

✅ Adicionar livros ao backlog  
✅ Marcar livros como lidos ou pendentes  
✅ Filtros por autor, categoria e status de leitura  
✅ Pesquisa dinâmica para encontrar rapidamente os livros desejados  
✅ Interface responsiva para dispositivos móveis

## 🚀 Como Rodar o Projeto

1. **Clone o repositório**:
   ```sh
   git clone https://github.com/seu-usuario/mybookhub
   cd mybookhub
   ```
2. **Instale as dependências**:
   ```sh
   npm install
   ```
3. **Execute o servidor de desenvolvimento**:
   ```sh
   npm run dev
   ```

O aplicativo estará disponível em `http://localhost:3000`.
